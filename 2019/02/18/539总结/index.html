codeforces Round 539(div 2)总结

发表于 2019-02-18

这场题简单的题做得都还开心，难题是真的自闭233333

A

题意：
给定数轴上有n个连续的点，坐标为1,2,3…,n。之后你驾车从1号点开始出发。一开始你的油箱为空，然后你可以在途中加油，油箱的容积为$v$,在点$i$加一升油的代价为$i$。问你最后至少需要多少代价才能到达终点。

数据量:$2 \le n \le 100,1 \le v \le 100$

分析:
感觉自己的贪心功力还是太弱了。简单题对着分析了一阵。既然越靠前的点我们的代价越低，那么我们就尽量在前面加满油，保证最后的$[n-v+1,n]$不需要加油即可。

B

题意:
给出n个数$a_1,a_2,a_3…a_n$,你可以将其中某个数$a_i$除以其某个约数$x$,并将其余的某一个数$a_j$变成其$xa_j$.问经过这样操作后这些数和的最小值是多少。

数据量:$2 \le n \le 5 \cdot 10^4,1 \le a_i \le 100$

分析:
注意到本题积一定下问和最小会是多少。因而我们无论怎么选缩小的数，扩大倍数的数必定是n个数中的最小的数。而对于如何选数还有倍数或许可以贪心(?)但由于本题数据量较小，因而我们选择枚举每个数的约数即可。复杂度为$O(n*max(a_i))$

C

题意：
给出$n$个数$a_1,a_2,a_3…a_n$,问有多少段长度为偶数的区间，使得前一半的数的异或值等于后一半的异或值。

数据量：$2 \le n \le 3 \cdot 10^5,0 \le a_i < 2^{20}$

分析:
异或值相等很容易转换成前缀和然后找前缀和相等且奇偶性相同的区间即可

因而我们可以在处理前缀和的时候建立前缀和-数量的键值对，然后遍历一遍记得到答案。注意零个数的前缀和为0.

D

题意:
给定一个回文串，问你至少将字符串分成多少部分，使得他们进行重新拼接后(切割后的字符串不能颠倒顺序)可以组成不同于原来字符串的新回文串。如果不能输出”Impossible”.

数据量:(记字符串为s,$1 \le |s| \le 5\,000$)

分析:

很有趣的一道题目。

首先我们考虑何时不可能。当字符串全为同一个字母，或者字符串为奇数个的时候，前缀$[1,…mid-1]$与后缀$[mid+1…n]$个数相等的时候显然不可能切出来。

再考虑最多需要分成多少部分。记最长的前缀为$[1…len]$,最长的后缀为$[n-len-1…n]$.显然前缀与后缀互为颠倒串(即$reverse(pref)=suff$)因而只要两个不相等即可。因而最多只需要分成三块。

那么有没有可能分成两块？ 我们不得而知，需要暴力去尝试了23333333

总之，回文串的题目要多利用回文串的对称性（奇串中心对称，偶串轴对称），去积极发现一些推论和结论，然后利用这些来作为线索破解题目。

F

题意：
给出四个数$n,m,a,b$,问你能生成多少棵n个顶点的带边权的树，满足从顶点a到顶点b的路径的边权和为m


数据量:$2 \le n \le 10^6,1 \le m \le 10^6,1 \le a, b \le n,a \neq b$,结果对$10^9+7$取模

分析：

先介绍Cayley’s formula:

给定一个图，n个顶点,其中包含一个k连通分量。则生成树的个数为：
$T_{n,k}=kn^{n-k-1}$

首先我们枚举从点A到点B的路径数量，很显然路径长度是小于等于min(n-1,m)的,记为$edge$

1.路径上共有($edge+1$)个点，除去A,B外还有$edge-1$个点，选出这些点并排列的方案数为$A_{n-2}^{edge-1}$

2.给$edge$条路径赋值使得其和加起来为m即为经典的小球投盒问题,方案为$C_{edge-1}^{m-1}$

3.对于不在这条链上的其余边，即为在$[1,m]$上选择一个权值，即为$m^{n-edge-1}$

4.此时图中的连通分量大小为$edge+1$,故生成树的个数为$(edge+1)*n^{n-edge}$

综上，对于固定的edge，生成树个数为上述四项的乘积

枚举edge累加即得答案
